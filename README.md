# Computational-Geometry

This is a Computational Geometry repository. This repository mainly covers geometric algorithms. The following topics are covered: basic geometric functions, triangles, polygons, Sweep Line, Bentley-Ottman, segment intersection, tangent lines, convex polygons, convex hull, Graham Scan, Gift Wrapping, Jarvis March, Andrews algorithm, incremental algorithm, divide and conquer, Polygon triangulation, Delunay, Point Cloud, Doubly Connected Edge List, DCEL.

Language used is Python. Folders uploaded are numerated, but usually one folder depends on the previous ones. It usually uses functions defined on previous files, and they are only explained the first time they appear.
