This is a Computational Geometry repository. This repository mainly covers geometric algorithms. 
The following topics are covered: basic geometric functions, triangles, polygons, Sweep Line, Bentley-Ottman, 
segment intersection, tangent lines, convex polygons, convex hull, Graham Scan, Gift Wrapping, Jarvis March, 
Andrews algorithm, incremental algorithm, divide and conquer, Polygon triangulation, Delaunay, Point Cloud, 
Doubly Connected Edge List, DCEL.

Language used is Python. Most files use functions defined on previous scripts. Example: we define `orient` in folder 1, and use it in 
most of the following projects.
